Research Projects

Conversational system framework Alex

We are working on an open-source conversational system framework called Alex. Our aim is to make the framework widely accessible and open in order to allow wider adoption of dialog systems in practice.

Demo of our system is running at 800 899 998 (you need to be able to speak Czech and be in the Czech Republic to be able to reach it).

Explicit Semantic Analysis

We applied Explicit Semantic Analysis for automatic and cross-lingual link discovery in document collections. Our work eventually placed in the top three places at the CLLD NTCIR-9 competition.

You can find our publication here.

I built a useful library (Java + C) esalib for experimenting with Explicit Semantic Analysis.

Affiliation

I am a PhD student at Institute of Formal and Applied Linguistics at Faculty of Mathematics and Physics at Charles University in Prague, Czech Republic. My research topic is modelling of complex statistical dialogue systems, and my thesis advisor is Filip Jurčíček.

Other Projects

  • ESAlib - out-of-the-box solution if you want to start experimenting with Explicit Semantic Analysis, as well as a supportive library to do more complicated things (indexing large corpora, using own background document collection); written in Java and C
  • videosucker - extracts (not only) youtube videos from .pcap packet capture files
  • th[eE]ater - our bot that took part in RedBot competition organized by Red Hat
  • MikroTik API for Python - Python Library that allows you control the MikroTik routers in your network from your scripts. Built atop SSH.
  • English to LTL Formula Translator (transLTLate) - Will transform your sentence into a LTL formula. It uses the typed dependency output of the Stanford parser and some rules to process the sentence and produce the formula. The formula can be used as an input for formal verification.
